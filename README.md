# BARKOD-OKUMA-PROGRAMI
Barkod Okuma ve Excel Raporlama Uygulaması 
(VB.NET Windows Forms) Barkod okuyucu üzerinden girilen verileri otomatik olarak algılayan; okunan değerleri anlık olarak listeleyen, her barkodun karakter uzunluğunu hesaplayan ve verileri sıralı şekilde DataGridView üzerinde gösteren bir masaüstü uygulaması geliştirilmiştir. Uygulama, toplanan verileri tek tuşla Excel formatında dışa aktarabilmekte ve oluşturulan rapor klasörüne hızlı erişim sağlamaktadır.
